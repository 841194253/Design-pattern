# Design-pattern


所写的部分代码来自于黑马的设计模式视频、CSDN、和C语言中文网

C语言中文网的代码为Java版本，所有统统改为cpp版本

部分代码不属于设计模式的一部分，属于黑马的部分课外设计代码运行无碍

所用的语言版本为C++11、CLion版本为2021.1、Cmake版本为3.19
